## Animation
- Transition property will change from initial state for the final state
- Duration

```css
button{
    padding: 30px 60px;
    background-color: blue;
    border: 0;
    color: white;
    transition-property: background-color;
    transition-duration: 500ms;
}

button:hover{
    background-color: green;
}

```
